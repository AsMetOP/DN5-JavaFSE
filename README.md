## 🧠 What Is This?

This is my solution repo for **Cognizant's Digital Nurture 5.0** — a 7-week intensive deep-skilling track covering the full Java enterprise stack, from raw DSA and design patterns all the way up to Spring Boot microservices, Angular, DevOps, and GenAI.

345 students were selected for this cohort. This repo is my proof of work.

Everything here is built **from scratch, by hand** — no copy-pasted boilerplate, no skipped exercises. If a pattern looks ugly in week 1, that's because I hadn't learned the clean way yet. That's the point of a learning log.

```
Design Patterns  →  DSA  →  Spring Core/JPA  →  Spring REST  →
Microservices    →  Angular  →  CI/CD  →  Docker/Cloud/GenAI
```

## ⚙️ How to Run Any Exercise

Each exercise is a standalone, dependency-free folder. No Maven needed for Week 1 pattern/DSA work — just the JDK.

```bash
cd Week1/DesignPatterns/ExX_PatternName
javac *.java
java <ClassNameWithMain>
```

For Spring Boot modules (Week 2 onward), each will ship its own `pom.xml` and run via:

```bash
mvn spring-boot:run
```

## 📜 Workflow

Every exercise follows the same loop:

```
Read exercise spec  →  Understand the WHY  →  Write the code  →
Compile & verify output  →  Commit  →  Push
```

Commit format used throughout this repo:

```
Week<N>: Ex<X> - <ConceptName> complete
```
